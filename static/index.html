Four ways to make a model
✦ Words → part: describe it, hit ✍ Spec, fix the ASSUMPTIONS lines, then Generate from spec. (Plain Generate skips the review.)
✦ Remix a model: attach a base via 📦 (file), 🔗 (Printables/Thingiverse link), then describe changes.
✦ Sculpt a photo: 🧬 + a photo of ONE object on a plain background → real 3D mesh on your GPU (1–3 min, stylized).
✦ Logo → part: drop a black-on-white logo image on 📦 — it traces to smooth curves you can extrude.

Buttons
📷 reference image for the AI (inspiration, not geometry) · 📦 base model file / logo · 🔗 import by URL · 🧬 photo→sculpture · number box = sculpt size in mm

After it generates
Sliders re-render live · type again + Refine = edit this model · New model = start over · STL single file · 3MF (parts) = separate colorable parts for AMS · → Bambuddy uploads to your archive.

Status line decoder
"vision check: fixed ×N" = the AI caught and repaired its own mistakes · "⚠ floating spots" = slice with tree supports or ask a refine to fix · "⚠ local fallback model" = codex is rate-limited, expect crude results — regenerate later.

Prompting tips
Name locations ("on the front face", "behind the wheel") · give real measurements when fit matters · batch related changes in one refine · after many refines, rebuild clean from the base in one big prompt · rate everything 👍/👎 — liked models teach future generations your style.
